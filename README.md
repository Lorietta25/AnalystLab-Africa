# AnalystLab-Africa

## Project Overview

This repository contains two end-to-end data analytics projects: Online Retail Store Analysis and Netflix Content Analysis. The projects showcase data cleaning, validation, exploratory data analysis (EDA), and interactive dashboard development using SQL and Power BI.

## Technologies & Tools

* SQL Server (SSMS)
* Power BI Desktop
* Microsoft Excel
* PowerPoint
* Git & GitHub

## Skills Demonstrated

* Data Cleaning & Validation
* Exploratory Data Analysis (EDA)
* Data Visualization
* Dashboard Design
* Business Intelligence Reporting

## Online Retail Store Analysis

Description: A retail transaction dataset used to analyze sales performance, customer purchasing behavior, product performance, and revenue trends.

## Key Insights

* Generated approximately $9.73M in revenue.
* Over 5.2M units sold.
* United Kingdom was the top revenue-generating country.
* Sales peaked in November.

## Files

* Online Retail Store dataset.csv
* Online retail file.sql
* Online Retail Store Dashboard.png

## Netflix Content Analysis

Description: A Netflix Movies and TV Shows dataset used to analyze content distribution, ratings, genres, countries, and release trends.

## Key Insights

* 8,807 titles analyzed.
* 6,131 Movies and 2,676 TV Shows.
* United States contributed the most content.
* Content releases peaked between 2018 and 2019.

## Files

* Netflix dataset file.csv
* Netflix dataset file.sql
